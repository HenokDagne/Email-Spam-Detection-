# Email Spam Detection Project

This project aims to build a logistic regression model to classify emails as spam or not spam.

## Structure
- `data/raw/`: Raw dataset from Kaggle
- `data/processed/`: Cleaned and preprocessed data
- `notebooks/eda.ipynb`: EDA, visualization, and preprocessing
- `src/`: Source code for data prep, modeling, and utilities
- `models/`: Saved model artifacts
- `outputs/logs/`: Log files
- `outputs/figures/`: Generated figures

## Requirements
See `requirements.txt` for dependencies.
